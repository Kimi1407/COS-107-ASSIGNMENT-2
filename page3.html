FIRST SEMESTER COURSES

This is a list of the courses offered by software engineering students in Babcock University during the first semester.

Course Title: General Physics
Course Code: PHY 101
This course provides essential knowledge in areas like mechanics, electromagnetism, and thermodynamics. It helps software engineers in simulations, algorithm optimization, hardware integration, and signal processing, ensuring better hardware-software interaction, efficiency, and realistic system design. This is a 2 unit course.
Course Title: General Practical Physics
Course Code: PHY 107
This course focuses on hands-on experimentation and real-world applications of physics principles. It helps software engineers understand physical systems and enhances problem-solving skills and aids in designing efficient, reliable software solutions. This is a 1 unit course.
Course Title: Use of Library and Study Skills
Course Code: BU-GST 105
This course helps students develop research skills, information literacy, and study techniques. It focuses on accessing academic resources and managing time effectively. This is a 2 unit course.
Course Title: Communication in English
Course Code: GST 111
This course enhances students' ability to convey ideas clearly and professionally. It focuses on improving written and verbal communication, technical writing, documentation, and collaboration skills which are essential skills to possess in the field. This is a 2 unit course.
Course Title: Elementary Mathematics
Course Code: MTH 101
This course builds a foundation in mathematical concepts like algebra, logic, and problem-solving. These skills are critical for understanding algorithms and data structures. This is a 2 unit course.
Course Title: Descriptive Statistics
Course Code: STA 111
This course equips students with tools to analyze and summarize data effectively. Key topics like measures of central tendency, variability, and data visualization help in software testing, performance analysis, and decision-making. This is a 3 unit course.
Course Title: Introduction to Computing Sciences
Course Code: COS 101
This course is foundational for software engineering students as it covers core concepts such as algorithms, programming basics, data structures, and computer systems. It provides the essential knowledge needed to design, develop, and analyze software solutions. This is a 3 unit course.
Course Title: Introduction to Scripting Languages
Course Code: BU-COS 107
This is essential for software engineering students as it focuses on lightweight programming languages like Python or JavaScript. It teaches automation and task specific coding. This is a 1 unit course.
